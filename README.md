# Rede de Influencers - Mockup

Mockup funcional da plataforma Rede de Influencers, conectando empresas a influenciadores verificados com foco em sustentabilidade e engajamento real.

## Visualização Online

Acesse o mockup: [Ver Mockup](https://anitaqa.github.io/rededeinfluencers/)

## Sobre o Projeto

A Rede de Influencers é uma plataforma inovadora que conecta empresas a influenciadores digitais verificados, com métricas reais de engajamento e compromisso com a sustentabilidade.

### Características Principais

- **Busca Avançada**: Filtros por categoria, localização, engajamento e sustentabilidade
- **Perfis Verificados**: Dados sincronizados com APIs oficiais (Instagram, YouTube, TikTok)
- **Sistema de Ranking**: Algoritmo transparente baseado em engajamento real
- **Sustentabilidade**: Sistema de níveis que valoriza ações ambientais
- **Negociação Direta**: Plataforma segura para propostas e pagamentos

## Como Usar

### Visualização Local

1. Clone o repositório:
```bash

```

2. Abra `index.html` no navegador

Use os botões flutuantes no canto inferior direito para:
- Download HTML (arquivo standalone)
- Download PDF (via impressão)
